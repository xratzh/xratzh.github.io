首先讲讲手里的条件：

  • 笔记本是T430i。i5-3380m配合金士顿骇客神条16G的ddr3-2133频率。独显nvs5400m和900P显示屏。
  • 穷
  • 有时间
  • Thinkpad有众多的折腾用户作为基础，良好的做工和可更换的硬件，使得折腾的人比较多，所以在网上很容易找到比较满意的适合自己电脑的硬件文件进行改造。

改造步骤：

先讲讲不完美的地方：

  • 首先是无线网卡无法驱动，毕竟网卡的话，苹果自家只认可博通，联想基本是螃蟹卡和Intel的无线卡，更换的话，需要换自己的网卡，但是还需要刷BIOS白名单才能使得硬件得以驱动。
  • 第二是miniDP可以使用，但是VGA接口无法驱动
  • USB3.0设备链接USB3.0接口的时候，关机按键变成重启，也就是说，要关机需要拔掉USB3.0设备。
  • NVS5400不能驱动，放弃了独显。

第一步

  • 下载以下的文件：
    TrandsMac、macOSwithClover.dmg、Clover configurator、HotPatch文件。
    链接: https://pan.baidu.com/s/1kUHQAqz 密码: vccc
    解压密码都是：itpwd.com
  • 准备一个8G的优盘，windows或者MacOS系统，这里只讲windows。其余的就是上面下载的文件。

第二步

把系统优盘准备好：
右键以管理员身份打开TransMac，在TransMac界面的左侧窗口找到需要制作启动盘的优盘，右键点击该优盘，选择菜单中的”Format Disk for Mac”格式化优盘，格式化完了在同样的位置右键单击该优盘，选择”Restore with Disk Image”，然后选择需要写入的镜像，确定格式化后等待完成就制作完毕。

第三步

使用上面Win平台制作的优盘插入你的电脑开始EFI引导安装吧

  • 1.引导选择上面制作的优盘，进入clover的引导界面，用方向键选择“Install macOS Sierra”这个磁盘确定进入，进入后就可以见到macOS的安装界面了，双击使用”磁盘工具”，选中需要安装到的本地硬盘，再点击“抹掉”全盘格式化需要安装到的硬盘，重命名为”macOS”；

  • 2.结束后关闭网页，选择安装到磁盘，经过比较漫长的等待，这个屏幕显示的剩余时间不准确。重启之后又在clover选择从install sierra进入sierra。开始短暂等待。之后又重启，正式进入系统开始配置

  • 3.这时引导文件还在优盘内，我们需要把引导文件也复制到本地的硬盘，要么每次进入系统都需要插入优盘，进入系统后打开Clover Configurator。
    点击左侧菜单的”Mount EFI”，在中下方的”Efi Partitions”内注意名字找到”macOS”所在的磁盘点击右边的”Mount Partiton”，再打开Finder工具左侧就能看到新挂载的EFI分区了，把需要导入的EFI文件夹拉进去就完成。这里先删除EFI文件夹。我们要加入patch里面合适的做好的EFI文件，具体是直接解压你合适的patch文件后，找到带EFI的那个文件夹，重命名文件夹为EFi并复制进去。

  • 4.以后调试EFI就在第3步的操作内，要是调试异常或者进不去系统了，需要把确定没问题的EFI引导放一份在优盘，有问题就用优盘引导就好。包括各种驱动patch都是在clover里面操作的。
