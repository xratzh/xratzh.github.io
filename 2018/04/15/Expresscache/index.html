原因：

很多的人都有一个疑惑，比如我有一个很小的固态硬盘，但是用作系统盘又实在是太小，想买个120G或者更大的固态，如今全部都是TLC的颗粒，素质也不是很好，好点的又很贵。如何物尽其用？那就是Expresscache，利用小的固态进行缓存加速方案的实施。

主要原理，是依据电脑上通过安装Expresscache，进行缓存，用户在安装后进行操作，常用的操作都会被缓存到固态盘里面，读取的时候就会优先对固态进行读取，以此使用的是固态的速度。达到一个提速的效果。

目前来看，总结了以下几个优劣的点供参考：

优点：

  • 成本低，物尽其用
  • 速度提升相较于机械的盘很大，达到一般的固态水平
  • 最重要的一点，由于固态只是起到缓存读取的作用，所以，基本数据依然是放在机械盘的，不必担心固态损坏造成的数据无法找回，机械盘的备份优势和修复优势体现出来了

缺点：

  • 小固态一般速度都不是顶级，但也是很优秀的水平
  • 固态比较小，所以缓存的内容比较少，一般是16G-64G的水平，推荐用于开机，玩游戏和浏览器等常用操作（似乎也没其他什么操作了

操作步骤：

如果你的笔记本在BIOS的Power选项里，支持Intel rapid start technology，那还可以通过固态起到快速开机的作用，一般的ThinkPad是有的，其他的没怎么见到。这个的话，暂时省略了。

1、首先是插上固态，通过diskpart清除固态的分区以及数据，然后重新启动，安装Expresscache，安装过程中，软件会自动把固态未格式化的分区给格式化，然后开启加速。

2、Expresscache下载地址 点我下载

如果你想要安装Intel rapid start technology，就先给Intel rapid start technology提供一个不大于你的内存的分区，然后重启，再安装这个Intel rapid start technology，最后再安装Expresscache。
